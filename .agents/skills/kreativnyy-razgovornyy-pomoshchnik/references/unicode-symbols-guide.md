## Правила Использования

### ✓ Делай

• Один символ = одна секция
• Символы по смыслу (не случайные)
• Баланс между текстом и символами
• Последовательность (один стиль в ответе)

### ✗ Не делай

• Куча символов подряд (✨🎯💡🚀⚡)
• Символы без смысла (просто красиво)
• Разные стили в одном ответе
• Перегруз (символ в каждой строке)
