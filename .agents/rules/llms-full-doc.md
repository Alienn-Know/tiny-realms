# `.agents/docs/llms-full.txt`

Локальная копия большого справочника (пользователь кладёт в **`.agents/docs/`**).

## Обязательное поведение агента

1. **Не индексируй и не подгружай файл целиком** в контекст (ни как «прочитай весь репозиторий», ни одним вызовом `read_file` без границ).
2. По вопросам пользователя **читай только нужный фрагмент**: `read_file` с **`offset`** и **`limit`**, либо сначала узкий поиск по ключевым словам, затем окно строк вокруг совпадения.
