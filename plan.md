# Интерактивная визуализация на TS + PixiJS — План

## Пошаговый план с нуля

### Фаза 1 — Основы (2-3 недели)

**Неделя 1: TypeScript**
- TypeScript Handbook — прочитай Basics + Interfaces + Generics
- Сделай 5 мини-задач: todo-лист, калькулятор, работа с API

**Неделя 2: PixiJS**
- PixiJS Getting Started
- Нарисуй: круг, прямоугольник, спрайт, текст
- Сделай: drag & drop, клик-события, анимация

**Неделя 3: Соедини вместе**
- Vite + TS + PixiJS проект
- Сделай интерактивную карту комнаты (двигай мебель мышкой)

### Фаза 2 — Первый проект (1-2 недели)

**Сделай визуализацию своих данных:**
- Выгрузи историю трат из банка (CSV)
- Построй интерактивную карту расходов: круги = категории, размер = сумма
- Клик по кругу → детализация за месяц

Это твой первый кейс в портфолио.

### Фаза 3 — Портфолио (2 недели)

Сделай 3 демо-проекта, залей на GitHub Pages:

1. **Конфигуратор** — кастомизация кружки (цвет, текст, шрифт)
2. **Дашборд** — визуализация данных (возьми open data с Kaggle)
3. **Интерактивная схема** — план офиса/склада с кликабельными зонами

### Фаза 4 — Первые деньги

1. Upwork → фильтр: "interactive visualization", "canvas", "PixiJS"
2. Откликайся с Loom-видео твоего демо (2 мин)
3. Первые 2-3 проекта бери за $500-1500 ради отзывов
4. После 5 звёзд → подними цену до $50-100/час

### Стек для старта

```bash
npm create vite@latest my-viz -- --template vanilla-ts
cd my-viz
npm install pixi.js
npm run dev
```

### Что НЕ делать
- Не учи React/Vue параллельно — сначала чистый TS + PixiJS
- Не делай "идеальный" проект — делай "рабочий и залитый онлайн"
- Не жди пока "будешь готов" — начинай откликаться на Upwork через месяц

### Ресурсы (бесплатно)
- **PixiJS** — pixijs.com/tutorials
- **TS** — totaltypescript.com/tutorials
- **Данные** — kaggle.com/datasets (бери любые CSV)
- **Дизайн** — dribbble.com/search/data-visualization (копируй идеи)

### Реалистичный таймлайн
- **Месяц 1** — учишь TS + PixiJS, делаешь 3 демо
- **Месяц 2** — первые фриланс-заявки, 1-2 мелких проекта ($500-1500)
- **Месяц 3-4** — стабильные заказы $2-5k/мес
- **Месяц 6+** — либо агентство, либо SaaS, либо найм $4-8k/мес

## Как продавать

**Шаг 1 — Найди боль**
- Зайди на сайт компании, найди их публичные отчёты/PDF
- Сделай бесплатный прототип их данных в интерактиве за 1 вечер

**Шаг 2 — Покажи разницу**
- Отправь Loom: "Вот ваш PDF-отчёт → вот интерактив. Кликните сюда."
- ROI аргумент: "Ваш аналитик тратит 4 часа на отчёт → с этим инструментом 10 минут"

**Шаг 3 — Ценообразование**
- Не продавай часы. Продавай результат
- "Внедрение интерактивного дашборда" = $20-50k fixed price
- Поддержка + обновления = $2-5k/мес
