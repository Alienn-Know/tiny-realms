# Tiny Realms

**Tiny Realms** — учебно-портфолио проект по созданию интерактивных визуализаций на **TypeScript + PixiJS**.

Этот репозиторий следует пошаговому плану развития от первых строк кода до первых фриланс-заказов. Полный план лежит в [`plan.md`](./plan.md).

## Цель проекта

Научиться создавать интерактивные canvas-приложения, собрать портфолио из 3 демо и выйти на фриланс-заказы в нише data visualization / interactive configurators.

## План в двух словах

| Фаза | Что делаем | Срок |
|---|---|---|
| **Фаза 1 — Основы** | TypeScript + PixiJS: фигуры, события, drag & drop, анимация | 2–3 недели |
| **Фаза 2 — Первый проект** | Интерактивная карта расходов по CSV из банка | 1–2 недели |
| **Фаза 3 — Портфолио** | 3 демо: конфигуратор, дашборд, интерактивная схема | 2 недели |
| **Фаза 4 — Первые деньги** | Upwork, Loom-видео демо, первые проекты | постоянно |

## Стек

- [TypeScript](https://www.typescriptlang.org/)
- [PixiJS v8](https://pixijs.com/)
- [Vite](https://vite.dev/)

## Быстрый старт

```bash
npm install
npm run dev
```

## Скрипты

- `npm run dev` — dev-сервер
- `npm run build` — сборка
- `npm run typecheck` — проверка типов

## Что дальше

1. Открой [`plan.md`](./plan.md).
2. Начни с **Фазы 1** — основы TS и PixiJS.
3. Делай демо, заливай на GitHub Pages, собирай портфолио.

## Статус

Проект на стадии прохождения плана. Текущая активная фаза и прогресс фиксируются в коммитах.
