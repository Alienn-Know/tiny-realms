# Tiny Realms

**Tiny Realms** — интерактивная песочница для изучения **TypeScript + PixiJS** и отработки навыков визуализации на canvas.

Этот репозиторий развивается по плану из [`plan.md`](./plan.md): от базовых фигур и событий до фриланс-заказов в нише interactive visualization.

## Что внутри

Сейчас проект находится на этапе базовых демо и отработки PixiJS-инструментов:

- **Камера** — pan/zoom по рабочему полю
- **NotebookPage** — пиксельное рабочее поле с масштабированием
- **FontShowcase** — демо битмап-шрифтов
- **ItemIconsLayer** — интерактивные иконки предметов
- **WaterSurface** — анимация воды через тикер
- **UIDemo** — пиксельные UI-элементы
- **ContextMenu + ClipboardPaste** — взаимодействие мыши и горячих клавиш

## Цель

Пройти путь от «нарисовал круг» до рабочих интерактивных визуализаций и первых фриланс-проектов.
