# Notes

Здесь собраны рабочие заметки по визуальному стилю, интерфейсу и техническим решениям проекта.

## Разделы

- `typography.md` - шрифты, тени, читаемость, pixel text rules.
- `ui-style.md` - стиль интерфейса, кнопки, панели, состояния, pixel-art принципы.
- `rendering.md` - рендеринг, `PixiJS`, слои, canvas-правила.
- `ideas.md` - быстрые идеи, гипотезы и наброски для будущих фич.

## Как Использовать

- новые заметки лучше добавлять в тематический файл;
- если тема новая, можно создать отдельный файл внутри `notes/`;
- короткие рабочие мысли, которые потом надо развить, можно временно складывать в `ideas.md`.
