# Паттерн "соло → $50M+" — разбор 3 кейсов

## 3 канонических кейса

| | **Stardew Valley** | **Undertale** | **Terraria** |
|---|---|---|---|
| Автор | Eric Barone | Toby Fox | Andrew Spinks |
| Возраст на старте | 23 | 19 | 25 |
| Год начала | 2011 | 2012 | 2010 |
| Год релиза | 2016 (5 лет) | 2015 (2.5 года) | 2011 (1.5 года) |
| Инструмент | C# / MonoGame | GameMaker | C# / XNA |
| Бюджет старта | $0 (подработка в кинотеатре) | $51K на Kickstarter | $0 |
| Продано | 30M+ копий | 5M+ копий | 44M+ копий |
| Доход | **$200M+** | **$50M+** | **$200M+** |

## 10 общих паттернов

### 1. Игра "против рынка", а не "для рынка"

> **Barone:** *"Я хотел сделать игру, от которой меня не тошнит"*
> *— реакция на Farmville с микротранзакциями*

> **Fox:** *"Я хотел сделать Earthbound, который я не могу перепройти"*

> **Spinks:** *"Просто хотел 2D-Minecraft, но который я сам бы играл"*

**Паттерн:** все трое делали игру, которую сами хотели бы купить. Не "что продаётся", а "что я люблю".

### 2. Стартовые инструменты — из нижнего сегмента

| Игра | Инструмент | Почему |
|---|---|---|
| Stardew | C# + MonoGame (не Unity) | Полный контроль, не платит royalties |
| Undertale | GameMaker (Pro) | Дёшево, быстрый prototyping, 2D-friendly |
| Terraria | C# + XNA (Microsoft) | Прямой доступ к пикселям |

**Не брали Unity/UE.** Брали **дешёвые, прямые, без royalty** инструменты. Это снижало зависимость от вендора и позволяло вечно владеть кодом.

### 3. 5+ лет упорной работы

- Barone: 5 лет в подвале родителей
- Fox: 2.5 года (но сожжённый, не выходил)
- Spinks: 1.5 года до релиза + **14 лет бесплатных обновлений**

**Ключ:** не "год работы → релиз", а "год работы → релиз → ещё 10 лет поддержки".

### 4. Self-funded / Kickstarter, НЕ publisher

```
Stardew:    подработка в кинотеатре, доход жены
Undertale:  Kickstarter ($51K от фанатов)
Terraria:   личные сбережения
```

**Никто не отдал IP.** Никаких "X даст 500K за 30% revenue". Когда игра выстрелила — **100% прибыли им**.

### 5. Personal obsession, не market research

Ни один из них не делал:
- Опросов "что бы вы хотели в игре?"
- Аналитики конкурентов
- GDD на 100 страниц
- Pitch deck'ов

Все делали:
- Личный vision
- Бесконечный self-iteration
- "Что бы я улучшил в своём любимом жанре"

### 6. Длинный хвост через бесплатные обновления

Terraria выпустила **~30 крупных бесплатных обновлений** за 13 лет. Каждое обновление = волна продаж (New & Trending в Steam, попадание в стримы).

**Почему работает:**
- Покупатели чувствуют благодарность → положительные отзывы
- Нет "выжали деньги" ощущения
- Маркетинг через годы бесплатный

### 7. Случайный вирус от стримеров

| Игра | Кто заметил |
|---|---|
| Stardew | Крупные ютуберы в 2016–2017 |
| Undertale | PewDiePie, инди-стримеры в 2015 |
| Terraria | Yogscast, многие LP-еры в 2011 |

**$0 на маркетинг.** Только качественный продукт + сообщество. Стримеры сами нашли и сами показали.

### 8. Mobile-порт = $50M+ чек

Все трое выпустили мобильные версии через **2–4 года** после ПК-релиза:
- Stardew Mobile (2018): $1M+ одному разрабу за портирование
- Undertale: $25 ticket
- **Terraria Mobile: $200M сделка с 505 Games** (Chengdu)

**Мобильный порт = второй релиз**, иногда приносящий больше, чем ПК.

### 9. Cut features, не добавляй

- Stardew: 8 skills вместо 20
- Undertale: 3 персонажа с глубокой историей вместо 20 поверхностных
- Terraria: 1 биом в первой версии (forest), добавили остальные позже

**Anti-pattern:** "Давайте добавим ещё, чтобы было больше". **Pattern:** "Вырежем всё, что не усиливает ядро".

### 10. Mental resilience: years of no income

- Barone: 5 лет работал без зарплаты
- Fox: ушёл из колледжа, жил на деньги родителей + KS
- Spinks: 4 года до того, как Terraria стала приносить больше зарплаты

**Финансовая подушка / side income обязательна.** Без неё не выживешь 2–5 лет.

## План "Solo → $50M+" на 5 лет

### Год 0: Подготовка (6 мес)

```
[ ] Выбери жанр, который ЛЮБИШЬ
[ ] Найди простые инструменты (GameMaker, Godot, MonoGame)
[ ] Сделай 1–2 мини-прототипа за выходные
[ ] Создай портфолио: 3–5 finished games
[ ] Накопай $20–40K подушки (2 года жизни на минимуме)
[ ] Создай публичное devlog в Twitter/YouTube/itch.io
```

**Не:** регистрируй LLC, ищи инвесторов, пиши GDD.

### Год 1: Vertical slice

```
[ ] Выбери 1 core mechanic (mining, farming, exploring, building)
[ ] Сделай её ИДЕАЛЬНОЙ за 3–6 мес
[ ] Сделай 1 уровень / 1 биом / 1 сценарий, который играется 30+ мин
[ ] Покажи 10 людям, получи фидбек
[ ] Итерируй, пока не зацепит
```

**Цель:** доказать себе, что механика работает и затягивает.

### Год 2: Полная версия v0.5

```
[ ] 5–10 часов контента
[ ] Базовый progression
[ ] 1–2 "wow moments" (открытия, боссы, твисты)
[ ] Саундтрек (найди композиторов на Fiverr / Discord)
[ ] Early Access на itch.io / Steam
[ ] Devlog каждую неделю
```

### Год 3: Public release + маркетинг

```
[ ] Steam page за 6 мес до релиза
[ ] Wishlist target: 50–100K
[ ] Trailer (сделай сам или найди $500–2000)
[ ] Demo доступна
[ ] Контакт с 50–100 стримерами/инди-блогерами
[ ] Релиз: $10–25 ticket
[ ] НЕ РАСПРОДАЙ СРАЗУ — дай органике набрать
```

### Год 4: Поддержка + экспансия

```
[ ] Большое обновление через 3–6 мес (бесплатное)
[ ] Перевод на 5+ языков (через сообщество)
[ ] Switch / console ports (найди портирующую студию)
[ ] Дорожная карта на год
[ ] Рост продаж: 5–10x от стартового
```

### Год 5: Mobile-порт = чек на $5–50M

```
[ ] Найди mobile porting studio (Dotemu, 505 Games, Playdigious)
[ ] Договорись: фикс + revenue share
[ ] Релиз iOS / Android
[ ] Re-reveal: "теперь на телефоне!" → второй пик
```

**К этому моменту у тебя $10M+ в кармане** (если игра успешная).

## Чего НЕ делать (anti-patterns)

| ❌ Антипаттерн | ✅ Что вместо |
|---|---|
| Писать GDD 100 страниц | Написать 1-страничный pitch и начать делать |
| Искать publisher | Self-publish + Kickstarter |
| Брать Unity/UE | GameMaker/Godot/MonoGame |
| "Сделаю игру для рынка VR mobile" | "Сделаю игру, в которую сам буду играть" |
| Команда 5 человек | Команда 1 (+ жена/друг) |
| Брать внешние инвестиции | Self-fund через side job |
| Free-to-play + ads | Premium $10–25 + бесплатные обновления |
| Мобильный-первый | PC-первый, mobile позже |
| Mid-development feature creep | Cut hard, polish what works |
| Год на маркетинг до релиза | 6 мес активной разработки → demo → релиз |

## Ключевой инсайт

**Toby Fox после Undertale** сказал:
> *"Я не ожидал ничего. Я думал, может, 100 человек купят"*

**Eric Barone:**
> *"Если бы я знал, что это будет популярно, я бы испугался и не закончил"*

**Andrew Spinks:**
> *"Я просто добавлял то, что сам хотел. Люди говорили, что я 'неправильно' веду игру. Я игнорировал."*

Все трое **сознательно не пытались сделать хит**. Они делали игру для себя. Хит — побочный эффект того, что они делали это годами без компромиссов.

### TL;DR формула

```
1 любимый жанр
+ 1 доступный инструмент
+ 1 core mechanic, отточенная 1000+ часов
+ 3–5 лет упорной работы
+ Self-funded, 100% IP
+ 0 инвесторов, 0 publisher
+ Бесплатные обновления годами
+ Случайный вирус от стримеров
+ Mobile-порт через 2–4 года
= $50–200M solo
```

**Вероятность:** ~0.1% игр зарабатывают $50M+. Но **$500K+ — реалистично для 1%**, и этого хватит на остаток жизни.
